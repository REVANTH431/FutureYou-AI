# 🚀 FutureYou AI – Simulate Your Future

> **“What if your future self could guide your present decisions?"**

FutureYou AI is an interactive AI-powered web application that helps users explore possible future outcomes based on their current choices. It combines modern UI design with intelligent logic to create a futuristic decision-making experience.

---

## 🌟 Project Vision

The goal of FutureYou AI is to bridge the gap between **decision-making and consequences** by simulating future scenarios.

Instead of guessing your future, you can **visualize it.**

---

## ✨ Features

* 🧠 AI-inspired decision simulation
* 🎨 Awwwards-level animated landing page
* ⚡ Smooth UI interactions & transitions
* 🌌 Futuristic design with immersive experience
* 📱 Fully responsive (mobile + desktop)
* 🚀 Built using modern frontend technologies

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/REVANTH431/FutureYou-AI.git
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the App

```bash
npm run dev
```

👉 Open in browser:

```
http://localhost:5173
```
